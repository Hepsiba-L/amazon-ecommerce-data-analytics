Project Overview

This project analyzes Amazon e-commerce sales data using Excel, Python, and Power BI. The goal is to transform raw data into meaningful insights that support better business decisions.

Project Workflow
1. Data Cleaning
Imported raw sales data.
Checked for missing values and duplicates.
Corrected data formats and standardized column names.
Cleaned inconsistent and invalid records to improve data quality.
2. Data Processing & Analysis
Performed exploratory data analysis (EDA) using Python.
Analyzed sales trends, product performance, customer behavior, and order patterns.
Calculated key performance indicators (KPIs) such as total sales, profit, orders, and average order value.
3. Data Visualization
Created interactive dashboards in Power BI.
Developed charts and visual reports to track sales performance.
Visualized trends by product category, region, and time period.
Used filters and slicers for dynamic analysis.
4. Business Insights

Key insights generated from the analysis include:

Identification of top-performing products and categories.
Analysis of sales trends across different periods.
Understanding customer purchasing patterns.
Detection of high-revenue regions and market opportunities.
Recommendations to improve sales performance and business growth.

Tools & Technologies
Excel – Data cleaning and preparation
Python – Data analysis and exploration
Power BI – Dashboard creation and data visualization

Outcome

The project demonstrates an end-to-end data analytics workflow, from data cleaning and analysis to visualization and business insight generation, helping stakeholders make data-driven decisions.
